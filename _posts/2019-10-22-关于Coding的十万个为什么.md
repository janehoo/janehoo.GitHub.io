- 【**Flutter**】问题：代码块写在setState(){}里面和外面有什么区别？
```
    // 示例1：代码块在setState(){}里面
onTap: (index) {
          setState(() {
            _selectedIndex = index; //代码块
          });
        },

    // 示例2：代码块在setState(){}外面
onTap: (index) {
          _selectedIndex = index;
          setState(() {});
        },
```
回答：没区别。就只是写在里面的时候会在Debug模式下检查代码块是否是同步的。
